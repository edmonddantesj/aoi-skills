# Security Policy (Core)

Non-negotiables:

1) Local-only analysis. Do not send scanned content off-machine.
2) Fail-closed outputs: PASS/WARN/BLOCK.
3) No auto-remediation by default (no auto-delete, no auto-edit, no auto-quarantine).
4) No automatic posting, submissions, or external messaging.
5) Never print secrets in output; only show safe excerpts / redacted matches.
