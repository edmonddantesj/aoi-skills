Status: CANONICAL
Scope: Aoineco & Co. squad-wide naming / dispatch / collaboration reference
Last updated: 2026-03-16
Primary source: Notion page `🧭 Aoineco & Co. Squad HQ` (`3049c616-de86-81a3-8420-e9a6188d96f6`)

## Purpose
업무 과중 토픽에 서브 팀원을 병렬 파견할 때,
팀원 명칭(한글명 / 영문명 / 역할)을 몰라서 호출·지시·배정이 흔들리는 문제를 막기 위한 canonical reference.

원칙:
- 팀원 명칭과 영문명은 Squad HQ 표 기준으로 해석한다.
- 파견/협업/지시/에스컬레이션 시 이름이 헷갈리면 이 문서를 먼저 본다.
- HQ와 상충하는 별칭/임시표현은 canonical이 아니다.

## Operating interpretation
### When a topic is overloaded
- 청령(Oracle)은 토픽 업무 과중/병목을 감지하면, 놀고 있는 다른 팀원을 적재적소·적시에 서브 파견해야 한다.
- 서브 팀원 파견 시, 해당 팀원이 그 토픽에서 작업 수행하는 데 문제 없도록 명칭/영문명/역할/범위를 함께 맞춘다.
- 파견된 서브 팀원은 이름/영문명/역할이 불명확하면 추측하지 말고 HQ / 본 문서를 기준으로 확인한다.

### Dispatch / mention rule
- 문서/공지/핸드오프에서는 가능하면 `한글명(영문명)` 형식을 사용한다.
- 역할 표기는 보조 설명으로 붙이고, 이름 축의 canonical source는 HQ 표다.
- writer/persona/profile 등 다른 문서에서 쓰는 별도 naming 축이 있더라도, HQ 표와 충돌하면 HQ가 우선이다.
