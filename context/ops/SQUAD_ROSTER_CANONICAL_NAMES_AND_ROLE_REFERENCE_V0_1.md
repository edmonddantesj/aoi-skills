# Squad Roster — Canonical Names & Role Reference V0.1

Status: CANONICAL
Scope: Aoineco & Co. squad-wide naming / dispatch / collaboration reference
Last updated: 2026-03-16
Primary source: Notion page `🧭 Aoineco & Co. Squad HQ` (`3049c616-de86-81a3-8420-e9a6188d96f6`)

## Purpose
업무 과중 토픽에 서브 팀원을 병렬 파견할 때,
팀원 명칭(한글/영문 역할명)을 몰라서 호출/지시/배정이 흔들리는 문제를 막기 위한 canonical reference.

원칙:
- 팀원 명칭과 역할은 Squad HQ 기준으로 해석한다.
- 파견/협업/지시/에스컬레이션 시 이름이 헷갈리면 이 문서를 먼저 본다.
- HQ와 상충하는 별칭/임시표현은 canonical이 아니다.

## Operating interpretation
### When a topic is overloaded
- 청령(CoS)은 토픽 업무 과중/병목을 감지하면, 놀고 있는 다른 팀원을 적재적소·적시에 서브 파견해야 한다.
- 서브 팀원 파견 시, 해당 팀원이 그 토픽에서 작업 수행하는 데 문제 없도록 명칭/역할/범위를 함께 맞춘다.
- 파견된 서브 팀원은 이름/역할이 불명확하면 추측하지 말고 HQ / 본 문서를 기준으로 확인한다.

### Dispatch / mention rule
- 한글 팀원명을 기본 canonical name으로 사용한다.
- 괄호 안에는 영문 역할명이 아니라 **English Name / Codename**을 붙인다.
- 문서/공지/핸드오프에서는 가능하면 `한글명(English Name)` 형태를 권장한다.
  - 예: 청령(Blue-Command), 청뇌(Blue-Brain), 청약(Blue-Med)
